Easiest path -- drive everything from your machine (chart already installed):

   ../run_suite.sh -n {{ .Release.Namespace }} --report

   It waits for the pods, auto-discovers the server RoCE IP (ip -br a), runs both
   clients (same-leaf + spine-crossing), then builds report.html and copies it to
   ./report locally.

Manual equivalent, if you prefer step by step:

1) Get the server's RoCE IP:
   oc exec -n {{ .Release.Namespace }} rdma-server -- ip -br -4 addr show   # take the net1 (Multus VF) address

2) Run the matrix from each client:
   oc exec -n {{ .Release.Namespace }} rdma-client-leaf  -- bash {{ .Values.script.mountPath }}/{{ .Values.script.fileName }} client auto <server_ip> same-leaf
   oc exec -n {{ .Release.Namespace }} rdma-client-spine -- bash {{ .Values.script.mountPath }}/{{ .Values.script.fileName }} client auto <server_ip> spine-crossing
   -> each writes {{ .Values.results.mountPath }}/<env>-<ts>/ {setup.json, bw/*.csv, lat/*.unsorted.txt + *.json}

3) Generate comparison graphs + report.html:
